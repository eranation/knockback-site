Tutorial: Knockback Inspector

Try the demo

Overview

This tutorial demonstrates how to use Knockback to create a tree view of potentially recursive models.

The application is built using Brunch to give it a standardized application structure. Brunch uses Node.js and Node Package Manager (npm) so it works on Windows, Mac, and Linux.

Building a Scaffold Project

  1. Install Node.js
  2. Install Brunch
    npm install -g brunch
  3. Create a new project: 
    brunch new knockback-inspector
  4. Build the new project
    cd knockback-inspector
brunch build
  5. Open 'public/index.html' to confirm the project is working. You should see a well-formatted Brunch Welcome! page
  6. Now we'll clean out everything we don't need:
    • Delete everything in the 'app/models' folder
  7. Now add Knockout.js and Knockback.js:
    • Put the latest knockout.js in 'vendor/scripts'
    • Put the latest knockback.js in 'vendor/scripts'
    • Update config.coffee to point to the new files. it should look like:
      files:
  javascripts:
    ...
    order:
      before: [
        'vendor/scripts/console-helper.js',
        'vendor/scripts/jquery-1.7.2.js',
        'vendor/scripts/underscore-1.3.1.js',
        'vendor/scripts/backbone-0.9.2.js',
        'vendor/scripts/knockout-2.1.0.js',
        'vendor/scripts/knockback-0.15.1.js'
      ]
  8. Build the project again and check that there are no errors in your console. Use 'brunch watch' to make sure all changes are automatically picked up
    brunch watch
  9. Update the main Template and View to make sure Knockback.js is working
    • Update 'app/views/templates' to:
      <div id="content">
  <h1 data-bind="text: title"></h1>
</div>
    • Update 'app/views/home_view.coffee' to:
      module.exports = class HomeView extends View
  id: 'home-view'
  template: template
  
  constructor: ->
    super
    
    @view_model = kb.viewModel(new Backbone.Model({title: 'Knockback Inspector'}))
    
  afterRender: ->
    ko.applyBindings(@view_model, @el)
    • Update the following selectors in 'app/styles/main.styl' to:
      #home-view
  width: 800px
  margin: 0 auto
  background-color: #F5E0FF
  
h1
  text-align: center
  font-size: 2em
  color: #fff
  border-radius: 5px
  background-color: #CC66FF
    • Refresh your browser page and you should see meaning we're ready to go!
