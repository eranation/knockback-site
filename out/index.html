Knockback

brings Knockout.js magic to Backbone

Code

Dependencies

Knockback requires Backbone (and therefore Underscore.js) and Knockout.js to be already loaded on the page.

Features

  • Localization
  • Knockout.js' bindings and automagical update
  • Backbone's sleek Model definition

Some background

Both Knockout and Backbone have their strengths and weaknesses, but together they are amazing!

At the same time you can have the strong ORM provided by Backbone and the dynamic view-models that Knockout.js offers.

BackboneKnockbackKnockout.js
Bullet-proof ORM☚Weak ORM
(De)Serialization☚Manual serialization
MVC☛MVVM
Manual views update☛data-bind based sorcery
